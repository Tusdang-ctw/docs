---
title: Breakout Radar
api: "GET /api/breakout-radar"
description: "Discover games gaining momentum based on visits and engagement"
---

Returns games ranked by a composite score combining visit momentum and player engagement, normalized across all tracked games. This surfaces games gaining traction beyond just CCU.

<RequestExample>
```bash
curl https://rolearn.dev/api/breakout-radar \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json
{
  "games": [
    {
      "place_id": 16732694052,
      "name": "Fisch",
      "creator": "FischDev",
      "genre": "Simulator",
      "score": 0.92,
      "visits": 2500000000,
      "likes": 1800000,
      "ccu": 125000
    }
  ]
}
```
</ResponseExample>
