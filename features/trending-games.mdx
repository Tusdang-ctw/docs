The Trending Games page surfaces the most popular Roblox games ranked by current concurrent users (CCU), updated every 5 minutes.

## How It Works

RoLearn ingests CCU data from the Roblox API in two tiers:
- **Tier 1**: Top 300 games by CCU — updated every 5 minutes
- **Tier 2**: 800 rotating games — full cycle every ~5 hours

Each game receives a **trend score** (0–1) based on its normalized CCU rank across all tracked games.

## API Endpoint

```
GET /api/trending?period=24h&genre=simulator&limit=50
```

See [API Reference → Trending](/api-reference/trending) for full details.
