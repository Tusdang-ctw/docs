## What can you do with RoLearn?

<AccordionGroup>
  <Accordion title="Discover trending games before they blow up">
    Our ML-powered trend detection analyzes CCU velocity, engagement ratios, and genre dynamics to surface breakout games early — often days before they appear on Roblox's own trending lists.
  </Accordion>
  <Accordion title="Forecast CCU and revenue">
    Prophet-based forecasting models predict future CCU trajectories for any tracked game. Revenue estimation combines gamepass analysis with player engagement data.
  </Accordion>
  <Accordion title="Find underserved genres">
    The Genre Opportunity Engine scores genres by saturation, growth rate, and revenue potential — helping you find the right niche before building.
  </Accordion>
  <Accordion title="Benchmark your game against competitors">
    Compare your game's CCU, engagement, and monetization metrics against genre averages and top performers.
  </Accordion>
  <Accordion title="Analyze updates with AI">
    Track how your game updates affect player counts, with AI-powered analysis of what worked and what didn't.
  </Accordion>
</AccordionGroup>

## Architecture Overview

RoLearn pulls live data from the Roblox API every 5 minutes, processes it through ML pipelines, and serves predictions via a REST API that powers the dashboard.

```
Roblox API → Celery Workers → PostgreSQL + Redis → FastAPI → React Dashboard
                ↓
         ML Models (LightGBM, Prophet, Gradient Boosting)
```

<Note>
  RoLearn tracks **50,000+ Roblox games** with CCU data ingested every 5 minutes. New games are discovered automatically via smart scanning.
</Note>
